# AI Communication Rules

## Response Rules

Return only requested artifacts.

Do not explain unless asked.

Do not generate documentation unless requested.

Do not generate sample data unless requested.

Do not regenerate existing files.

---

## Coding Rules

Modify only requested files.

Preserve existing architecture.

Do not refactor unrelated code.

Do not create duplicate utilities.

Reuse existing components.

---

## Flutter Rules

Generate only affected widgets.

Avoid generating entire application.

Avoid placeholder screens.

Use existing theme.

---

## Backend Rules

Generate only requested endpoints.

Reuse existing middleware.

Reuse existing services.

Do not modify database schema unless requested.

---

## Output Rules

Maximum Response:

300 lines

For large tasks:

1. Plan
2. File List
3. Implementation

Stop after implementation.

---

Ignore deprecated requirements.

Never infer new requirements.
